<h2>📰About</h2>
<h3 align="center">
  Our idea is to create a School Events & Notification Center that makes organizing and joining school activities simple, fair, and automated. Through event creation, registration, and a smart waitlist system, students and organizers can efficiently manage participation while an event-driven notification system keeps users informed in real time.
</h3>
